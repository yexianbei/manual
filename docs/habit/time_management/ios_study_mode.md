# iOS 学霸模式帮助

## 什么是学霸模式？
学霸模式是一种能在专注计时期间帮助您减少其他App的干扰，从而提升专注效率的功能。学霸模式能尽量帮助您减少外界干扰，但专注过程中也需要您自己进入状态哦。
